Introducción a los Punteros

Los punteros son una de las características más poderosas y fundamentales de C++. Nos permiten manipular directamente la memoria del computador, crear estructuras de datos dinámicas y escribir código más eficiente.

Un puntero es una variable que almacena la dirección de memoria de otra variable.

¿Qué es Desreferenciar?

Desreferenciar un puntero significa acceder al valor almacenado en esa dirección de memoria.

Analogía:

El puntero es como una llave, y desreferenciar es quitar la referencia para abrir la caja y ver el dato.

Estructura de un Nodo

struct Nodo {
    int dato;
    Nodo* siguiente;
    
    Nodo(int valor) : dato(valor), siguiente(nullptr) {}
};

Ventajas de las Listas Enlazadas:

  • Tamaño dinámico
  • Inserción y eliminación eficiente
  • Uso eficiente de memoria

Ejemplos Prácticos

Operaciones Básicas con Punteros

// Declaración y inicialización
int x = 10;
int* ptr = &x; // ptr almacena la dirección de x
// Desreferenciación
cout << "Valor de x: " << x << endl; // 10
cout << "Dirección de x: " << &x << endl; // 0xFED4B2
cout << "Valor de ptr: " << ptr << endl; // 0xFED4B2
cout << "Valor apuntado: " << *ptr << endl; // 10
// Modificación a través del puntero
*ptr = 20; // Cambia el valor de x a 20
cout << "Nuevo valor de x: " << x << endl; // 20

Implementación de Lista Enlazada

// Definición de la clase Lista
class Lista {
private:
    Nodo* cabeza;
public:
    Lista() : cabeza(nullptr) {}
    void insertar(int valor) {
        Nodo* nuevo = new Nodo(valor);
        nuevo->siguiente = cabeza;
        cabeza = nuevo;
    }
    void mostrar() {
        Nodo* actual = cabeza;
        while (actual != nullptr) {
            cout << actual->dato << " -> ";
            actual = actual->siguiente;
        }
        cout << "NULL" << endl;
    }
};

Conceptos Clave para Recordar

Punteros

  • & - Operador de dirección
  • * - Operador de desreferenciación
  • ptr - Almacena dirección de memoria
  • *ptr - Accede al valor almacenado

Listas Enlazadas

  • Nodo - Elemento básico (dato + enlace)
  • Cabeza - Primer nodo de la lista
  • nullptr - Indica fin de lista
  • Dinámicas - Crecen según necesidad
