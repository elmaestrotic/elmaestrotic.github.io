¿Qué es Haskell?

Lenguaje Funcional Puro

Haskell es un lenguaje de programación de propósito general netamente funcional, diseñado 100% en el paradigma funcional.

Homenaje a Haskell Curry

Su nombre está dado en honor a Haskell Bruce Curry, matemático y lógico estadounidense que ayudó a sentar las bases de los lenguajes de programación funcionales.

Concepto de Currificación

El concepto de "curry function" también se hace en honor a su apellido, siendo fundamental en programación funcional.

Programación Funcional

Paradigma basado en funciones matemáticas puras y evaluación lazy

Instalación en Windows

Método recomendado usando Chocolatey

1

Visitar haskell.org

Ir al sitio oficial haskell.org y hacer clic en Download

Buscar la sección "Show Windows package" y seleccionar Chocolatey

2

Ejecutar PowerShell como Administrador

Abrir PowerShell con permisos de administrador para ejecutar los comandos de instalación

Windows PowerShell → Clic derecho → Ejecutar como administrador

3

Configurar Políticas de Ejecución

Verificar que las políticas permitan la instalación (debe aparecer "Bypass", nunca "Restricted")

Set-ExecutionPolicy Bypass -Scope Process -Force

4

Instalar Chocolatey y Haskell

Ejecutar los comandos de instalación

# Instalar Chocolatey

[System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

# Instalar Haskell

choco install ghc cabal stack

5

Verificar Instalación

Probar que la instalación fue exitosa

# Abrir símbolo del sistema (cmd) y ejecutar:

ghci

# Probar operación aritmética:

10 + 20

# Resultado: 30

Instalación de Stack

Gestor de dependencias para proyectos Haskell

¿Por qué Stack?

  • Administra dependencias automáticamente
  • Evita problemas de compatibilidad
  • Gestiona versiones de paquetes
  • Similar a Gradle en Android

Instalación

Ruta de instalación:

Creación de Proyectos

Línea de Comandos

1. Crear nuevo proyecto

stack new mi-proyecto simple

2. Configurar proyecto

cd mi-proyecto

stack setup

3. Construir proyecto

stack build

4. Ejecutar proyecto

stack exec mi-proyecto-exe

IntelliJ IDEA

1. Importar proyecto

File → Open → Seleccionar carpeta del proyecto

2. Configurar SDK

Seleccionar "Haskell Stack" como gestor

3. Configurar ejecutable

Especificar la ruta de Stack como ejecutable

4. Crear configuración

Run → Edit Configurations → Add New → Haskell

Tip: La IDE detecta automáticamente proyectos Haskell y configura el entorno

Estructura de Proyecto

Archivos Principales

stack.yaml

Archivo de configuración de dependencias y versiones del proyecto

src/

Carpeta con el código fuente principal (archivos .hs)

Main.hs

Archivo principal con la función main del programa

test/

Carpeta para pruebas unitarias y especificaciones

Plantilla "Simple"

-- Main.hs

module Main where


main :: IO ()

main = putStrLn "Hola mundo"

Características:

  • • Estructura básica para comenzar
  • • Configuración mínima necesaria
  • • Ejemplo "Hola mundo" incluido
  • • Lista para desarrollo inmediato

Funciones en Haskell

Sintaxis y Declaración

1. Firma de Función

-- Declaración del tipo

suma :: Int -> Int -> Int

Define que la función recibe dos enteros y devuelve un entero

2. Definición de Función

-- Implementación

suma x y = x + y

Define la operación que realizará la función

3. Función Main

main :: IO ()

main = do

  let result = suma 2 3

  putStrLn $ "La suma es: " ++ show result

Utiliza la monada IO para entrada/salida

Ejemplo Completo

-- Spec.hs

suma :: Int -> Int -> Int

suma x y = x + y


main :: IO ()

main = do

  let result = suma 2 3

  putStrLn $ "La suma es: " ++ show result

Compilar y Ejecutar

# Compilar

ghc Spec.hs


# Ejecutar

./Spec


# Salida: La suma es: 5

Conceptos Clave:

  • • let: Declaración de variables locales
  • • ++: Operador de concatenación
  • • show: Convierte a string
  • • $: Evita paréntesis innecesarios
